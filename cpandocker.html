NAME

cpandocker - easy cpan docker image maintain utility

SYNOPSIS

cpandocker [OPTIONS] action [cpan] [cpan, ...]

DESCRIPTION

This is the cpandocker tool that allows to make a perl sandbox and maintain what cpan modules are in there via a configuration file. The configuration file location and/or cpandocker home can be specified on command line.

cpandocker acts as a wrapper around docker. Building is done with the aardbeiplantje/perl and aardbeiplantje/perl:5.32.0-dev-latest docker images.

OPTIONS

help|h|?

Show the usage information.

man|m

Show the manpage.

ARGUMENTS

action

What to do for the list of cpan modules on the command line. Note that there doesn't have to be a cpan module specified, as there are actions that just list or manage the .cpandocker.json config file.

  • list

    list what cpan modules are dockerized as configuration.

  • cfgsync

    fetch the latest information from cpan and sync this with the config.

  • info

    fetch the cpan information from cpan for the modules that are added to the configuration

  • add

    add a cpan module, note that this also allows upgrading

  • build

    just build a module, don't add it to the configuration

  • rebuildall

    rebuild all cpans that are configured

  • version

    show the version of a configured cpan, this also acts as a perl module load check

  • versionall

    show the version for all configured cpans, this also acts as a perl module load check for all modules that are configured.

  • push

    push a configured and build module to the remote docker repository

  • pushall

    push all cpans that are configured to the remote docker repository

  • buildimage

    builds the end docker image

cpan

The cpan module to execute the action for
